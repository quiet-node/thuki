# Thuki

The context-aware floating secretary.

## Features

- **Context-Aware**: Understands your workflow and provides assistance when needed.
- **Floating Interface**: Accessible from anywhere on your systems with a clean, unobtrusive design.
- **Secure & Private**: Runs locally on your machine with minimal external dependencies.

## Development

This application is built using [Tauri v2](https://tauri.app) and [React](https://react.dev).

### Prerequisites

- [Rust](https://www.rust-lang.org/tools/install)
- [Node.js / Bun](https://bun.sh)

### Commands

- **Run Dev**: `bun run tauri dev`
- **Build**: `bun run tauri build`
